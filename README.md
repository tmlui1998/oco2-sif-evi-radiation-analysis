### OCO-2 SIF, EVI, and Solar Radiation Analysis
This repository contains a Python workflow for analyzing the relationship between solar-induced fluorescence (SIF), vegetation activity (EVI), and solar radiation (SSR) using satellite data. The pipeline integrates OCO-2 SIF observations, Google Earth Engine raster products, and spatial aggregation techniques to study vegetation productivity across Southern Finland.

The workflow processes raw satellite files, constructs spatial datasets, and applies Generalized Additive Models (GAM) to investigate nonlinear relationships between plant fluorescence, vegetation greenness, and incoming solar radiation.

### Data Sources
## OCO-2 SIF
Solar-induced fluorescence observations from the OCO-2 satellite.
